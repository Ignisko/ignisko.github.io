# Ignisko's Personal Website

This is the personal website of Ignacy Januszek, built using the Astro framework.

## Features

- Fast and optimized for performance
- Responsive design
- Supports Markdown content

## How to Use

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`

## Content

All posts are located in the `src/content/posts` directory. Add your Markdown files there to create new posts.

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions.
